# Horny Toad -- OpenClaw Agent Testing

A satirical dating-app-style testing ground for the OpenClaw agent framework. Currently running 30 autonomous frog agents with matchmaking logic, match modal displays, and a live ticker. Primarily a vehicle for testing agent autonomy, behavior composition, and data-driven registries.

Active development. 10+ commits today across 3 phases: basic layout, split-body hop mechanics with wink/blink/tongue animations, and 30-agent matchmaking with live ticker.
