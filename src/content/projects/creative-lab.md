# Creative Lab

A bleeding-edge playground running Next.js 16 with React Compiler. Six sectors of experimental UI. `ScrollVelocityContext` is the core physics engine — a custom context that tracks scroll velocity and provides it to all child components for physics-based animations.

## Technology Decisions

**Next.js 16 + React Compiler.** Chosen specifically to stay on the bleeding edge. Not production code — this is a learning ground for upcoming React features.

## Status

Experimental. No deploy pressure. Used for testing new patterns before they reach production projects.
